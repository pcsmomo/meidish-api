# meidish-api

## Development Steps

1. init npm and typescript and setup tsconfig.ts

   ```json
   // tsconfig.json
   {
     "target": "ES2016",
     "module": "commonjs",
     "rootDir": "./src",
     /* "moduleResolution": "node", */
     "outDir": "./build"
   }
   ```

2. Install Expresst

3. Install Mongoose and set it up

   - mongodb atlas

4. Connect to AWS S3 for images
